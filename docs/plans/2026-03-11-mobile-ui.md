# Mobile UI Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Add a responsive mobile layout that keeps the workspace primary and moves the library and preview into slide-over drawers on small screens.

**Architecture:** Extend `App` with small responsive UI state for the two drawers and pass mobile actions into `Header`. Keep existing `TagLibrary`, `Workspace`, and `Preview` components reusable by rendering them in desktop sidebars and mobile drawers with shared styling hooks. Cover the behavior with responsive rendering tests in Vitest + Testing Library.

**Tech Stack:** React 19, TypeScript, TailwindCSS 4, Vitest, Testing Library
